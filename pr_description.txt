🧪 Add tests for BlockRegistry schemas and instances

🎯 **What:** This PR adds missing test cases for `MemoryBlockSchema` and `MemoryBlock` within `BlockRegistry`, addressing a gap in testing coverage.

📊 **Coverage:** The new tests cover:
- `MemoryBlockSchema.validate_content` (char limit, custom validator, valid inputs)
- `MemoryBlockSchema.to_dict` and `MemoryBlockSchema.from_dict` (serialization and deserialization)
- `MemoryBlock.update_content` logic across `MergeStrategy.REPLACE`, `APPEND`, and `SYNTHESIZE`
- `MemoryBlock.is_empty` validation
- `MemoryBlock.to_dict` serialization

✨ **Result:** Test coverage for the `foresight_mcp/block_registry.py` components is significantly improved, adding testing confidence to serialization, validation, and merge strategies.
